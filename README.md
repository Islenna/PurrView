# PurrView

PurrView is an innovative project aimed at revolutionizing the way pet owners collect diagnostic-quality ophthalmic images. Leveraging the latest in AI technology and direct collaboration with veterinary specialists, PurrView offers a user-friendly platform for capturing high-quality eye images of your beloved pets.

## Introduction
PurrView is both a web and mobile application designed to facilitate the collection of diagnostic quality ophthalmic images. With an emphasis on ease of use and accessibility, PurrView ensures that pet owners can capture vital health information, which can be critical for veterinary diagnosis and care.

## Features
- **AI Learning:** Utilizes cutting-edge AI technology to guide users in capturing the best quality image, similar to taking a picture of a check through your bank's app.
- **Doctor-led:** Developed in direct collaboration with DVM, DACVO to ensure the images meet diagnostic standards, making it easier for your veterinarian to diagnose and treat eye conditions.

## Technologies
PurrView is built with a robust stack of technologies to ensure a seamless user experience across web and mobile platforms:
- **Frontend:**
  - **React Native:** A JavaScript framework for building native mobile apps.
  - **Expo:** Tools and services for React Native development.
  - **React Navigation:** Navigation solution for React Native apps.
  - **TypeScript:** Adds static typing to JavaScript to improve development and performance.

- **Backend:**
  - **MongoDB:** A NoSQL database for storing app data.

- **Machine Learning:**
  - **TensorFlow:** An open-source framework for machine learning and artificial intelligence.
  - **Keras:** A high-level neural networks API, utilized for developing and training machine learning models.

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- Node.js
- npm (Node Package Manager)
- Expo CLI (for Expo projects)

### Installation
1. Clone the repository:
    ```git clone https://github.com/your-username/purrview.git```

2. Navigate to the project directory:
   1. ```cd purrview```
3. Install the dependencies:
   1. ```npm install```
   
## Usage
To use PurrView, follow these steps:

1. Open the app on your web or mobile device.
2. Follow the on-screen instructions to capture an image of your pet's eyes. The AI assistant will guide you through the process.
3. Review the captured images and select the one you believe is the clearest.
4. Submit the image through the app for analysis.
